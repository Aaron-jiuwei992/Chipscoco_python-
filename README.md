## 1. 摘要

## 2. 题库简要概述
在这个题库中，我将存放一些由易到难的练习题，这些题往往十分基础，但是却是练习算法的最好选择。

## 3. 为什么要创建这个题库？
我将这个练习题册子当作自己平时练习题的存放仓库，方便以后随时查看和复习。
